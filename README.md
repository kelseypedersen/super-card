# Super Card

An application to learn more about Views. 


## Key Learnings
```
- Use of CGFloat, CGPoint, CGSize, and CGRect
  - scale factors
  - rotation of the suit/rank
- UIGestureRecognizer
  - connecting view to viewController
  - other types of gestures
- Subviews & Superviews
  - layering views
```
